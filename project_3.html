Dinosaur Listing

Project brief

In this project the team was tasked with making a website that displays a collection of dinosaurs using PhP, and object-oriented programming. The team focused on keeping the code clean and maintainable and making the website AA accessible. We're required to use design patterns such as Hydrators and View helpers, and it all had to be in OOP. We took on 5 stories and got them all done and nearly finished a 6th.
